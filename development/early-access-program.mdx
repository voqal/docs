# Voqal Assistant (EAP)

The Early Access Program (EAP) is a way for you to get access to the latest features and improvements before they are released.
This allows you to try out new features and provide feedback to help shape the future of the product.

## How to Join

Plugins -> Manage Plugin Repositories -> Add Repository
- https://eap.voqal.dev/plugin-repository

## Feedback

If you have any feedback or suggestions, please let us know by creating an issue on
[GitHub](https://github.com/voqal/voqal/issues)
or chatting with us on [Discord](https://discord.com/invite/KgTkR5Rffz).
