The Add Breakpoint tool is used to add breakpoints to the source code. Developers can specify lines numbers or other
natural language instructions to add breakpoints to.

### Example Usage

- `"Add breakpoints to all the system out print lines"`
- `"Add breakpoints to lines 5 and 10"`
- `"Add a breakpoint to every getter function in this class"`

### Source Code

```yaml
type: function
function:
  name: add_breakpoint
  parameters:
    type: object
    properties:
      line_numbers:
        type: array
        description: The line number(s) to add a breakpoint to.
        items:
          type: string
    required:
    - line_numbers
  description: Adds breakpoints to the specified line numbers.
```
