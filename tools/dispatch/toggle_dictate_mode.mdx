### toggle_dictate_mode
```yaml
type: function
function:
  name: toggle_dictate_mode
  parameters:
    type: object
    properties:
      prompt:
        type: string
        description: The raw transcript of the user's speech
    required:

  description: Pass transcription off to Voqal Dictate Mode. Use this tool whenever
```
