Voqal provides a wake word detection feature that allows you to wake up the voice assistant by saying a specific word.
Alternatively, you can also wake Voqal up via the push-to-talk action. The default push-to-talk shortcut is `Ctrl-Shift-S`
, but can be changed in the IDE keymap settings.
